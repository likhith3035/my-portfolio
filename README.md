Welcome to the **Vibe Coder** experience. This isn't just a portfolio; it's a statement. Designed with a striking neo-brutalist aesthetic, high-performance animations, and zero lag.

## 🚀 Features

- **🎨 Neo-Brutalist Aesthetic:** Bold colors, harsh shadows, and intentional glitch effects.
- **🤖 Built-in AI Chatbot:** A fully integrated smart assistant that answers recruiter questions about my experience, skills, and projects in real-time.
- **📱 60FPS Mobile Optimization:** Intelligent rendering drops heavy effects (like SVG grain and complex parallax) on mobile devices to ensure a buttery smooth experience.
- **⚡ Framer Motion Powered:** Scroll-linked animations, magnetic buttons, and a physics-based custom cursor.
- **✉️ Serverless Contact Forms:** Direct integration with FormSubmit for reliable message delivery without a backend.
- **🔍 Ultimate SEO Setup:** Full JSON-LD schema, Open Graph tags, robots.txt, and noscript fallbacks ensuring top-tier Google ranking.

## 🛠️ Tech Stack

- **Framework:** React + Vite
- **Styling:** Tailwind CSS (with highly customized utility classes)
- **Animations:** Framer Motion
- **Icons:** React Icons
- **Deployment:** Vercel / Netlify automatically via GitHub Actions (CI/CD)

## 💻 Local Development Setup

Want to run the Vibe Coder experience locally?

**1. Clone the repository**
```bash
git clone https://github.com/likhith3035/my-portfolio.git
cd my-portfolio
```

**2. Install dependencies**
```bash
npm install
```

**3. Start the dev server**
```bash
npm run dev
```
Open `http://localhost:5173` in your browser.
